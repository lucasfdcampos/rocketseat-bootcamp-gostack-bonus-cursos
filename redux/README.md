## :gear: Configuração para desenvolvimento

```bash
  # clone o repositório
  $ git clone https://github.com/lucasfdcampos/rocketseat-bootcamp-gostack-bonus-cursos.git

  # acesse o diretório
  $ cd redux

  # instale as dependências
  $ yarn 

  # inicie o server JSON-server
  $ yarn server

  # inicie o projeto
  $ yarn start
```
